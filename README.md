# Sentiment_Analysis_
This project performs sentiment analysis on text reviews using Natural Language Processing (NLP) and Machine Learning. It includes text cleaning, tokenization, stopword removal, lemmatization, TF-IDF vectorization, and model training using Linear SVM. Visualizations like word clouds and word frequency plots are used to better understand the data.

Tech Stack: Python · NLTK · Scikit-learn · TF-IDF · SVM · Matplotlib
